## ⚠️ Troubleshooting

| Problem | Solution |
|---------|----------|
| Port in use | Change PORT in .env |
| npm error | `npm cache clean --force` |
| Python missing | Install from python.org |
| Upload fails | Check backend running |
| Blank page | Clear cache (Ctrl+Shift+Del) |
